# heart-disease-prediction
Heart Disease Prediction using Machine Learning

## Objective
Use medical characteristics to predict the risk of heart disease.

## Dataset
Heart Disease UCI Dataset

## Techniques Used
- Data Cleaning
Exploratory Data Analysis (EDA)
- Logistic Regression
- Feature Scaling

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC Curve
- ROC-AUC Score

## Results
- Accuracy: 81.97%
- ROC-AUC: 0.927

## Tools
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-Learn
